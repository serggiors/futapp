API REST
Nodejs express sequelize MySQL

/api/ > ruta principal
users (usuarios) /api/users/create/ > POST > crear registro de usuario
/api/users > GET > ver todos los usuarios
/api/users/id > GET > mostrar registro por id de usuario
/api/users/id > DEL > eliminar registro por id del user
/api/users/userId/bookings/create/ > POST > crear reserva por id de usuario
/api/users/userId/bookings/ > GET > mostrar todas las reservas por id de usuario
roles (roles) /api/roles/create > POST >crear rol
/api/roles > GET > ver todos los roles
/api/roles/id > GET > mostrar registro por id del rol
/api/roles/id > DEL > eliminar registro por id del rol
images (imagenes) /api/images/create > POST > subir url imagen
/api/images > GET > ver todas las url de imagenes
/api/images/id > GET > mostrar url por id de imagen
/api/images/id > DEL > eliminar url por id de imagen
peoples (personas) /api/peoples/create > POST > crear persona
/api/peoples > GET > ver todos los registros de personas
/api/peoples/id > GET > mostrar persona por id
/api/peoples/id > DEL > eliminar persona por id
soccerfields (campo de futbol) /api/soccerfields/create > POST > crear cancha de futbol
/api/soccerfields > GET > ver todas las canchas
/api/soccerfields/id > GET > mostrar cancha por id
/api/soccerfields/id > DEL > eliminar cancha por id
bookings (reservas) /api/bookings/create > POST > crear reserva
/api/bookings > GET > ver todas las reservas
/api/bookings/id > GET > mostrar reservas por id
/api/bookings/id > DEL > eliminar reserva por id
